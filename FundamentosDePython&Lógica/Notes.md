## Operadores lógicos

### Negação not

Ele retorna o inverso da expressão.

not True
>> False

not False
>> True

not 1000 > 1500
>> True.

porque ele verifica primeiro o 1000 > 1500, respondendo com false, e em seguida o python vê o operador not e inverte de false para true.
